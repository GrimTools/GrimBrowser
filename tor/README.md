# Tor binaries (per-OS)

Grim loads the Tor binary for your OS automatically:

- `tor/win/tor.exe`   — Windows (bundled)
- `tor/mac/tor`       — macOS   (add from the Tor Expert Bundle to enable Tor)
- `tor/linux/tor`     — Linux   (add from the Tor Expert Bundle to enable Tor)

Get binaries at https://www.torproject.org/download/tor/ . Without a binary for your OS,
Tor routing is simply disabled — everything else works normally.
